# Avery Carter Â· Teacher Portfolio

A polished, single-page teacher portfolio built with Next.js 16, the App Router, and Tailwind CSS. The site highlights classroom experience, teaching philosophy, professional learning offers, and resources, with a contact form that redirects to a thank-you screen.

## Features

- Hero narrative with key highlights and calls to action
- Classroom portfolio cards and teaching philosophy showcases
- Milestone timeline, testimonials, and professional learning offerings
- Resource library links and a contact form powered by FormSubmit
- Thank-you confirmation page for smoother outreach flow

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS (via `@import "tailwindcss";`)

## Local Development

```bash
npm install
npm run dev
```

The app runs at [http://localhost:3000](http://localhost:3000). Tailwind JIT updates styles instantly during development.

## Production Build

```bash
npm run build
npm run start
```

## Deployment
